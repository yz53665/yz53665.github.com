基于 SSH 连接（命令行操控）

  1. 创建一个空白文件，命名为 wpa_supplicant.conf

  2. 文件中输入一下内容

    country=CN
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1

network={
    ssid="******"
    psk="*******"
}

    主要关注 network 中的内容，ssid 是 WiFi 名，psk 是 wifi 地址。此外可以设
    置多个待选连接网络，但需要在 network 中额外增加一行 =priority=1=，里面的数
    字代表连接的优先级。

  3. 将设置好的文件放入树莓派储存卡的 boot 盘下即可，然后重新启动，树莓派就会按
    照事先设定好的网络自动进行连接。
  4. 将主机和待连接设备接入同一局域网下（同一个路由器）
  5. 登录路由器控制中心网页，进入 DHCP 界面查看树莓派 IP 地址

  6. 在命令行中输入：

    ssh [userNanme]@[ip address]

    其中树莓派默认用户名是 pi

  7. 按照提示输入密码即可成功连接

通过 VNC Viewer 连接（有图形界面）

  1. 在主机上安装 VNC Viewer

  2. ssh 远程连接树莓派，安装 tightvncserver ，具体命令如下：

    sudo apt-get install tightvncserver
  3. 在命令行输入 tightvncserver 激活 vnc，然后会提示设置登录密码，回车后就会
    成功开启，然后会提供一个端口号。
  4. 在主机中打开 VNC viewer ，输入 [ip]:[portName] ，就可以连接到树莓派。
