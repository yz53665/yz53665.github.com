1 code blocks 结构

简化形式：

src_<language>{<body>}  or
src_<language>[<header arguments>]{<body>}

2 基础设置

几个基本参数设置：

  • org-edit-src-auto-save-idle-delay
    设置每隔多长时间自动保存代码块中的内容
  • org-edit-src-turn-on-auto-save
    将代码块中的内容自动保存到单独的文件中
  • org-src-window-setup
    设置 c-c ’ 后代码编辑窗口的位置
  • org-src-preserve-indentation
    设置导出和 tangled 是否保留还是重新设置代码的缩进，如果设置成 nil，导出后缩
    进会被重新安排，非 nil 设置时会保留原本 codeblocks 中展示的缩进

4.1 python 如何便捷输出图片结果？

可以进行如下设置:

#+begin_src: python :results output file :session [运行环境名] -n -r
print('picture.png')
#+end_src
  • [运行环境名]：类似 python 这样的语言可以将不同的代码块运行在同一共享数据的
    持久化环境中，这里可以设置该环境的名字。之后的代码块只要设置了相同的名字就
    可以共享数据

  • :results output file
    设置为 output 后所有代码块中打印的内容都会在结果中显示出来， file 则说
    明所有输出对象是对已存在的文件的引用，自动将输出转化为 [file: 文件名]
    的格式，这样插入图片后可以通过 org-display-inline-images 在代码块下面显示
    图片，实现类似 jupyter notebook 的效果
    演示如下：

    import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['Hiragino Sans GB'] # 用来正常显示中文标签
plt.rcParams['axes.unicode_minus'] = False # 用来正常显示负号 

ts = pd.Series(np.random.randn(1000),
               index=pd.date_range("1/1/2000", periods=1000))
ts = ts.cumsum()

ts.plot()
plt.savefig('picture.png')
print('picture.png')
